# Local File Distributor Setup Guide

This guide helps you set up and make the most out of your Local File Distributor (LFD) portal.

## Quick Start
1. Place folders/files inside the shared folder.
2. Launch `run.bat` or run `python app.py`.
3. Open `http://localhost:5000` on the host PC.
4. Copy the Access URL or let clients scan the QR code.

## Access Password
You can lock the portal from the Host Panel:
- Enable "Password Lock".
- Type a secure access code (e.g. `math101`).
- Clients must enter this code to download files.
